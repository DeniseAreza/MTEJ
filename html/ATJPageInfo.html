AUTOMATIC THOUGHT JOURNAL

Ang Automatic Thought Journal ay isang feature kung saan sistematikong tinatanong ang mga users tungkol sa kanilang mga iniisip gamit ang gabay ng mga serye ng mga tanong. Nagbibigay-daan din ito sa mga users na pamahalaan ang kanilang interpretasyon ng mood at kung paano lulutasin ang kanilang mga triggering events. Ang journal ay isang tool na tumutulong sa pagtukoy sa mga automatic negative thoughts na lumilikha ng mood swings at pinoproseso ang kanilang mga damdamin sa pamamagitan ng gabay ng mga tanong. Ang mga tanong ay nakaayos sa paraan na makatutulong sa madaliang pag-unawa sa paraan ng kanilang pag-iisip at pagproseso sa kanilang mga damdamin.

PAANO GAMITIN ANG AUTOMATIC THOUGHT JOURNAL?

Upang higit na maunawaan ang paggamit ng Automatic Thought Journal, tatanungin ng aplikasyon ang mga users ang kanilang mga naging Negative Triggering Event, Automatic Thoughts, Automatic Feelings, at ang kanilang naging Resulting Behavior. Pagkatapos sagutan at mga hininging impormasyon, tatanungin ng web application ang mga user para naman sa kanilang mga Alternative Thought, Alternative Feeling, at Alternative Behavior, kung saan magpapatuloy ito sa pag-save sa database pagkatapos ng entry ng mga users. Gayunpaman, kung hindi nakaranas ng anumang triggering even and user, maaring nang tapusin ang proseso ng web application.
